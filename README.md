## Running FoodAhead
Follow the below steps to enable suggest macros.

1. Create a USDA FoodData Central API key:
   - Go to the official FoodData Central API key signup page.
   - Sign up for a free API key.

2. Add the API key to a `local.properties` file in the project root:

   USDA_API_KEY=your_api_key_here

3. Sync Gradle.

4. Run the app on an emulator or Android device.
